This is a Vite + React + Tailwind frontend that shows Strava segments on a Leaflet map.

Setup

1. Install dependencies

```bash
cd frontend
npm install
```

2. Run dev server

```bash
npm run dev
```

The dev server proxies `/api` to `http://localhost:4000` (see `vite.config.js`). Start the backend first.
